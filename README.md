# Fashion Store

A modern and responsive fashion e-commerce frontend built with **React + Vite**. This project was developed as part of the **Oxivos Frontend Developer Round 1 Project Task**.

## Project Overview

Fashion Store is a frontend-only e-commerce application that provides a smooth shopping experience using local dummy data. Users can browse products, view product details, filter by category, and manage a shopping cart—all without a backend.

---

## Features

- Modern Home Page with Hero Banner
- Product Listing Page
- Category Filtering
- Product Details Page
- Shopping Cart
- Add to Cart

- Remove from Cart
- Dynamic Total Price Calculation
- Product Ratings
- Color & Size Display

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/fashion-store.git
```

Go to the project directory

```bash
cd fashion-store
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build project

```bash
npm run build
```
